# PolyScope - Polymarket Trader Intelligence Platform

## Overview

PolyScope is an enterprise-grade AI-powered analytics platform designed to evaluate Polymarket traders based on their wallet addresses. The platform provides comprehensive trader intelligence reports, performance metrics, and data-driven follow recommendations.

## Purpose

In the rapidly evolving prediction market ecosystem, identifying skilled traders to follow is critical for both passive investors and active market participants. PolyScope addresses this need by providing:

- **Objective Trader Evaluation**: Removes emotional bias from follow decisions
- **Quantitative Analysis**: Uses proven statistical methods to assess performance
- **Risk Assessment**: Identifies potential risks before you commit to following
- **Actionable Insights**: Delivers clear recommendations backed by data

## Key Features

### 1. Follow Score (0-100)
A proprietary scoring system that combines five key metrics into a single, easy-to-understand score:
- Consistency Score
- Risk Management Score
- Accuracy Score
- Volatility Assessment
- Discipline Score

### 2. Intelligent Recommendations
Three-tier recommendation system:
- **FOLLOW**: High-confidence traders with strong track records
- **CAUTION**: Moderate performers requiring additional due diligence
- **DO NOT FOLLOW**: High-risk profiles or poor performers

### 3. Comprehensive Analytics
- Performance radar charts
- Cumulative PnL tracking
- Trade-by-trade analysis
- Strengths and weaknesses identification

## Target Users

- **Passive Investors**: Looking to mirror successful trader strategies
- **Active Traders**: Seeking to benchmark against top performers
- **Research Analysts**: Conducting trader behavior research
- **Risk Managers**: Evaluating counterparty trading competence

## Technology Stack

- **Frontend**: React + TypeScript + TailwindCSS
- **Backend**: Supabase Edge Functions
- **Data Source**: Polymarket API
- **Visualization**: ECharts

## Getting Started

1. Navigate to the PolyScope platform
2. Enter a valid Polymarket wallet address (0x...)
3. Wait for the AI analysis to complete
4. Review the comprehensive intelligence report
5. Make informed follow decisions based on data
